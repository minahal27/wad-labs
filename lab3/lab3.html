Report

In this lab task, I created a personal webpage to introduce myself, my interests, and my thoughts on web development. The webpage includes a horizontal navigation bar, sections for "About Me," "Likes," and "Dislikes," and a final "Report" section.

What I Did:

  • HTML Structure: I used semantic HTML tags like <nav>, <div>, <h1>, <h2>, <p>, <ul>, and <ol> to structure the content. The <nav> tag was used to create the navigation bar, while <div> tags were used to group related content.
  • CSS Styling: I styled the navigation bar horizontally using CSS. I used properties like display: flex, justify-content: center, and padding to align and space the links evenly. I also added hover effects using transition and :hover to improve interactivity.
  • Content Organization: I organized the content into sections using headings (<h1>, <h2>) and lists (<ul>, <ol>) to make it easy to read and navigate.

Why I Chose These Tags and Styles:

  • Semantic Tags: I chose semantic tags like <nav> and <div> to make the code more readable and meaningful. .
  • Flexbox for Navigation: I used Flexbox for the navigation bar because it provides a simple and efficient way to align items horizontally and center them within the container.
  • Hover Effects: I added hover effects to make the navigation bar more interactive and visually appealing. This enhances the user experience by providing feedback when hovering over links.

What I Learned:

  • I learned how to structure a webpage using semantic HTML tags and how to style it using CSS.
  • I gained a better understanding of CSS Flexbox and how to use it to create responsive and aligned layouts.
  • I learned how to improve user interaction by adding hover effects and transitions.
  • This task helped me understand the importance of organizing content into sections and using headings and lists to improve readability.
